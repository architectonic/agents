# Agents Gates

## Promotion ladder

```text
idea -> template -> archetype -> installed-example -> validated-dist -> published
```

## Required checks

- JSON files parse.
- Archetype bundles include identity, doctrine, skill bundle, models, knowledge sets, prompts, and upkeep.
- High-stakes archetypes default to low autonomy.
- Local doctrine narrows but does not weaken upstream doctrine.
- Skill pins preserve provenance.
- Knowledge attachments declare scope, privacy, freshness, and review requirements.

## Human approval gates

- Publishing packages.
- Increasing autonomy above level 3.
- Adding real client/private corpora.
- Claiming professional authority.
- Adding external mutation permissions.
